# 教育心理学知识库说明

## 知识库定位
本系统面向：
- 学生
- 家长
- 一线教师
- 学校心理支持团队

知识库用途包括：
- 教育心理学基础概念解释
- 学习动机、学习习惯、考试焦虑等常见问题支持
- 家校沟通与学校心理支持流程说明
- 风险识别与转介边界提醒

## 知识层次

### 1. 结构化知识层
来源于整理好的教育心理学与学校心理支持知识包。
特点：
- 有明确 `id / category / topic / risk_level / source_*`
- 适合直接进入向量索引
- 适合做问答、标签过滤、风险控制

### 2. 原始证据层
来源于教材、手册和原始 PDF 文档。
特点：
- 保留原文页级内容
- 适合补充证据和溯源
- 适合后续做更精细的章节切分

## 当前推荐使用策略
- 面向日常问答时，优先召回结构化知识层
- 当用户需要更详细背景或原文依据时，再结合原始证据层
- 对高风险话题，必须使用保守、支持性和转介导向的回答策略

## 高风险主题
以下内容必须触发更强的安全约束：
- 自伤
- 自杀
- 伤害他人
- 持续失眠伴明显功能受损
- 严重抑郁样表现
- 校园欺凌中的高风险伤害

## 当前已接入的资料类型
- 教育心理学与学校心理支持结构化知识包
- 大学教师教学手册 PDF 页级抽取
- 教育心理学教材 PDF 页级抽取
- 普通心理学教材可保留为后续补充原始资料
